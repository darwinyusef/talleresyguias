# Comando: /refactor

Refactoriza código para mejorar su calidad y mantenibilidad.

## Argumentos:
- `target` (opcional): archivo o directorio específico a refactorizar

## Pasos:

1. Identifica código que necesita refactorización:
   - Duplicación de código
   - Funciones muy largas
   - Complejidad ciclomática alta
   - Código mal organizado
2. Propone cambios de refactorización
3. Explica los beneficios de cada cambio
4. Implementa los cambios si el usuario está de acuerdo
5. Ejecuta tests para verificar que no se rompió nada
