# Comando: /review

Revisa el código reciente y sugiere mejoras.

## Pasos:

1. Identifica los archivos modificados recientemente
2. Analiza el código en busca de:
   - Bugs potenciales
   - Problemas de seguridad
   - Oportunidades de optimización
   - Violaciones de mejores prácticas
   - Problemas de estilo/formato
3. Proporciona sugerencias específicas con ejemplos
4. Prioriza las sugerencias por impacto
